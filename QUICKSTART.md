### üßÆ Using the Calculator

#### Basic Operations
1. **Click buttons** or **use keyboard** to enter numbers
2. **Click operator** (+, -, √ó, √∑) or **press operator key**
3. **Click = or press Enter** to calculate
4. **Click C or press Escape** to clear

#### Moving the Calculator
1. **Click and hold** the purple header bar
2. **Drag** to your desired position
3. **Release** to drop

#### Keyboard Shortcuts
```

Decimal:    . (period)
Add:        + (plus)
Subtract:   - (minus)
Multiply:   * (asterisk)
Divide:     / (forward slash)
Equals:     Enter or =
Clear:      Escape or C
Backspace:  Backspace
```

### üé® Using the Whiteboard

#### Drawing
1. **Click and drag** on the white canvas to draw
2. **Choose color** with the color picker
3. **Adjust size** with the slider (1-20px)

#### Erasing
1. **Click "Eraser"** button (turns purple when active)
2. **Draw over content** to erase
3. **Click "Eraser"** again to switch back to pen mode

#### Adding Text
1. **Click "Text Mode"** button (turns purple when active)
2. **Click anywhere** on the canvas
3. **Type your text** in the prompt
4. **Click OK** to place the text

#### Clearing Canvas
- **Click "Clear"** button to erase everything

## üí° Tips & Tricks

### Calculator Tips
- **Chain calculations**: Type `5 + 3 + 2 =` without clearing
- **Decimal precision**: Use `.` for decimal numbers
- **Quick clear**: Press `Escape` key for fast clearing

### Whiteboard Tips

- **Vary thickness**: Change pen size for emphasis

- **Color coding**: Use different colors for organization

### General Tips

## ‚ö†Ô∏è Troubleshooting

### Calculator won't move
- Make sure you're clicking and dragging the **purple header** (not buttons)

### Whiteboard not drawing
- Check if **Eraser mode** is active (button should be dark)
- Make sure you're **clicking and dragging** (not just clicking)

### Keyboard not working
- Make sure **no text input is focused**
- Try clicking on the background first

### History not saving
- Check if **browser allows localStorage**
- Try a different browser if issues persist

### Colors not changing
- Make sure **Customize panel is open**
- Click the color button for the element you want to change
